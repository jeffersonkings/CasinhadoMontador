# Casinha do Montador

Aplicação web para gestão de serviços de montagem, com painéis personalizados para clientes, profissionais e administradores.

## Funcionalidades

- Cadastro e login com autenticação Firebase
- Painel do cliente com histórico, notificações e agenda
- Painel do profissional com serviços disponíveis, em andamento e concluídos
- Painel do administrador com filtros e gestão de solicitações
- Gráficos de desempenho semanal e mensal
- Upload de documentos e validação de CPF

## Tecnologias

- React
- Firebase (Auth, Firestore, Storage)
- Chart.js
- React Router
- CSS Modules
